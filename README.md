# DungeonCrawler Tester

Testing utilities and mock encounters for DungeonCrawler testing.

## Features

- Test encounter generators
- Mock character creation
- Automated test fixtures
- Unit and integration test utilities
- Performance testing tools

## Installation

```bash
composer require --dev forseti-life/dungeoncrawler-tester
drush pm:enable dungeoncrawler_tester
```

This module is primarily for development and testing purposes.
